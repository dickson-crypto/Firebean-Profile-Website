## 1. What This System Does

This is a **weekly EDM** (Electronic Direct Mail) series called **"PR MARKET DECODED"** by Firebean Limited, targeting Hong Kong Government officers and procurement decision-makers on a cold list. The EDM must feel like a professional industry briefing — value-first, not promotional.

**Why weekly production is manual (not bulk):** Section 01 contains 2 real, current HK government news items that must be sourced fresh each week from official .gov.hk websites. These cannot be pre-generated.

## 4. Weekly Production Workflow (Step-by-Step)

### Step 1 — Source 2 Fresh HK Gov News Items (S01)

Search these official sources for current exhibitions, public programmes, or government campaigns:
- `https://www.lcsd.gov.hk/en/` — LCSD programmes and exhibitions
- `https://www.museums.gov.hk/en/web/portal/home.html` — All LCSD museums
- `https://hk.history.museum/` — HK Museum of History
- `https://hk.science.museum/` — HK Science Museum
- `https://www.hkpl.gov.hk/` — HK Public Libraries
- `https://www.info.gov.hk/gia/` — Government press releases
- `https://www.icho.hk/` — Intangible Cultural Heritage Office
- `https://www.ekcc.hk/` — East Kowloon Cultural Centre

**For each news item, you need:**
- Official name (bilingual if available)
- Category tag (e.g. GOVERNMENT · EXHIBITION, GOVERNMENT · FAMILY)
- Source organisation (e.g. "Hong Kong Museum of History (LCSD)")
- 1 thumbnail photo — **download it**, convert to JPG, commit to `edm/email-img/` as `[descriptive-name].jpg`
- 1-sentence italic punchline
- 1 body paragraph (60–80 words)
- Official "Learn More" URL (must be a live .gov.hk or lcsd.gov.hk URL)

**Image rule:** Never hotlink government/venue images. Always download → convert to JPG → commit to `edm/email-img/` → reference via GitHub Pages URL: `https://dickson-crypto.github.io/Firebean-Profile-Website/edm/email-img/[filename].jpg`

---

### Step 2 — Select the Case Study (S03)

Pick the next unused **Government project** from the Master DB Google Sheet:
`https://docs.google.com/spreadsheets/d/1Ms1Q1i7uJg0ilvW4g1PezBm7mTCNKcYJT_c5-weUBNc/edit`

- Filter `Category = GOVERNMENT & PUBLIC SECTOR`
- Sort by `Event Date` descending (most recent first)
- Skip any Project_id already used in previous issues (see EDM_Database tab)

**For the case study, you need:**
- Project_id (e.g. `FB2026048`)
- Client name
- Project name
- Scope of Work
- Hero banner photo: download from Drive folder → convert to JPG → commit to `edm/email-img/` as `[project_id_lowercase]-photo-[N].jpg`
- Thumbnail photo (different from hero): same process, different photo number
- Case study title (8–12 words, action-outcome style)
- Punchline (1 italic sentence, 15–20 words)
- Body content (150–200 words: Challenge → Solution → Key Outcomes with 3 → bullets)
- Profile URL: `https://firebean.net/profile.html?id=[project_id_lowercase]`

**Drive photo URL formula:**
1. Get folder ID from "Drive Folder Link" column in Basic Info sheet
2. List files: `gws drive files list --params '{"q": "[FOLDER_ID] in parents and mimeType contains \"image/\"", "fields": "files(id,name)"}'`
3. Find `photo_N.jpg` matching "Hero Photo Picker" column
4. Download: `https://drive.google.com/uc?id=[FILE_ID]&export=view`

### Step 3 — Select/Confirm Market Insight (S02)

Use the pre-researched insight from the EDM_Database tab (column Z–AD) for this issue number, OR research a fresh one if generating a new issue beyond #24.

**For a fresh insight:**
- Topic must be unique — not covered in any previous issue
- Must include a real statistic with a named source (MarketIntelo, McKinsey, Deloitte, HKSAR Policy Address, academic journal, etc.)
- 150–180 words with 3–4 → bullet implications for HK government departments
- AI-generated image: 16:9, professional editorial, navy blue/teal palette, no text overlay
- Upload image to CDN or commit to `edm/email-img/`

### Step 5 — Generate Gmail Version

```bash
cd edm/
python3 build_legacy_gmail.py edm_NNN.html
```

The script will:
- Remove VML/MSO conditional comments
- Convert hero banner background-image to stacked `<img>` + dark overlay div
- Remove `float:right` from thumbnails (stacks vertically in email clients)
- Strip `object-fit` from inline styles
- Convert `.webp` URLs to committed `.jpg` equivalents
- Add `<style>` block with mobile breakpoints and bullet pseudo-elements
- Update pre-header href to GitHub Pages URL
- Run QA checks and report any issues
